My Final Project

Here on this page you will see the program I created called Alien Escape. It's a fun and interactive game that pushes your competitiveness. This program consist of the lessons learn throughtout the semester in CSC 106 course. Ideally the object of this game is to move the playable character up and down using the arrow keys between the aliens until you reach the end to win. If you hit an alien, its game over and you'll have to restart the game and play again. There are two levels to select, an easy mode and hard mode. Enjoy!
